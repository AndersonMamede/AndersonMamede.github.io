Resumo

Configurao

Interface Configurao Resumo.

  • Atributos
    • Ttulo : Permite informar um Ttulo para o resumo, caso no seja informado nenhum nome para este campo o scriptcase admite o ttulo padro para o resumo, assim o ttulo ficar RESUMO, porm se desejar que nenhum ttulo seja exibido ser necessrio informar na propriedade ttulo a tag HTML &nbsp;. Campo Ttulo sendo informado com a Tag HTML.
    • Ttulo Quantidade : Ttulo da quantidade de registros.
    • Quantidade de Registros :
      • No mostrar : No mostrar o total de registro por quebra.
      • Mostra apenas no resumo : Mostra no resumo o total de registros por quebra.
      • Mostra no resumo e no grfico : Mostra no resumo o total de registros por quebra e disponibiliza grfico.
    • Total Horizontal : Exibir total horizontal para resumos do tipo matriz.
    • Posio do cone do grfico : Posicionamento do cone do Grfico (esquerda ou direita).
    • Posio do cone do total : Posicionamento do cone Total (esquerda ou direita).
    • Total Vertical : Exibir total vertical para resumos do tipo matriz.
    • Largura do Resumo : Valor da largura para o resumo.
    • Unidade da Largura do Resumo : Unidade de medida usado para a largura. Automtico (valor da largura ignorado), pixels e porcentagem.

Configurao do Layout

Interface Configurao Resumo.

  • Atributos
    • Campo da quebra : Campos da quebra selecionados na consulta.
    • Posio : Define entre posio do eixo x ou do eixo y.
    • Ordenao : Define a ordenao pelo valor de banco ou pelo valor de exibio.
    • Preencher labels vazios : Define se os labels vazios vo ser preenchidos.
    • Link Grid : Cria um link no campo selecionado.
    • Alinhamento : Define o tipo de alinhamento do layout para centralizado, esquerda ou direita.
    • Formato tabular : Define o resumo com o formato tabular.

Barra de ferramentas

Neste quadro podemos fazer as configuraes referentes ao resumo. O resumo ocorre sempre que a aplicao de consulta contm exibir resumo, tipo de resumo, posio do resumo, ordenao do resumo, tecla de atalho, quantidade de registros, tipo do grfico, largura do grfico, altura do grfico, margem do grfico, aspecto do grfico, grfico em outra janela, largura do resumo e unidade da largura do resumo.

Interface Configurao do Resumo.

  • Atributos
    • Formatos : Permite selecionar os formatos de gerao de resumo.
    • Botes : Permite selecionar os botes que sero exibidos na barra de ferramentas.
    • Ordenao dos Botes : Permite ordenar os botes na barra de ferramentas conforme necessrio, utilizando a barra de seleo (lado direito da tabela).

Selecionar Campos

Nesta interface se definem os campos que sero totalizados e seu tipo de totalizao.

Interface Campos do Resumo.

Sero listados apenas os campos dos tipos Numeral, Decimal e Moeda. Para atribuir uma operao a ser utilizada numa coluna (campo), basta selecionar o tipo de totalizao desejada, marcando o Checkbox logo abaixo.

  • Soma : Define que seja gerado uma soma dos valores referente ao campo selecionado.
  • Mdia : Calcula a mdia aritmtica dos valores referente ao campo selecionado.
  • Mximo : Exibe o maior valor identificado no campo selecionado.
  • Mnimo : Exibe o menor valor identificado no campo selecionado.
  • Mdia ponderada : Exibe o clculo realizado pela mdia ponderada no campo selecionado.
  • Peso de mdia ponderada : Opo para escolha do peso utilizado no clculo da mdia ponderada.

Os campos selecionados sero exibidos quando ocorrer a quebra ou no final da consulta.

Posicionamento

Nesta interface podemos configurar o posicionamento dos campos da totalizao.

Interface posicionamento dos campos do Resumo.

Arraste e solte as opes para a posio desejada.

Group Label

Interface do Group Label do Resumo

Com essa opo, voc pode inserir um ou mais ttulos para suas colunas em suas aplicaes, podendo alterar seu tamanho, sua cor, sua fonte e posiciona-los como desejar.

Caso voc tenha campos com exibio dinmica o Group Label no ir funcionar.

Edio de propriedades da clula do group label

  • Atributos
Ttulo Permite definir o ttulo que ser exibido em seu Group Label.
Fonte de texto Permite definir a fonte do ttulo de seu group label.
Tamanho da fonte Permite definir o tamanho da fonte do ttulo de seu group label.
Alinhamento horizontal Permite definir o alinhamento horizontal do ttulo de seu group label. Sendo esquerda, centro ou direita.
Alinhamento vertical Permite definir o alinhamento horizontal do ttulo de seu group label. Sendo meio, baixo ou topo.
Cor de texto Permite definir a cor do texto do ttulo de seu group label.
Cor de fundo Permite definir a cor de fundo do espao onde o ttulo de seu group label ser exibido.

Ordenao

Interface Ordenao dos campos do Resumo.

  • Atributos
    • Campos : Selecione os campos que voc deseja permitir a ordenao.
    • Ordenao Inicial : Selecione um campo para ordenao inicial no resumo.
    • Tipo de ordenao : Escolha se a ordenao ser ascendente ou descendente.
    • Ordenar a partir da Quebra : Selecione uma das quebras para ordenao inicial.

Grficos

Neste quadro podemos informar as configuraes dos Graficos. Como por exemplo: tipo do grfico, largura, altura, margens, etc.

Se existir duas ou mais Regras de Quebra, devemos configurar as opes de grficos para cada Regra de Quebra criada.

O grfico do tipo Analtico s funciona quando se utiliza 2 campos configurados nas quebras. Se o usurio configurar mais de um campo, o grfico analtico s ir gerar com os dois primeiros.

Interface Gerao dos Grficos de Coluna.

Interface Gerao de Grficos de Linhas (Totais).

Estas configuraes esto disponveis somente para Regras de Quebras com mais de 1 campo.

Configurao dos grficos

Interface de Configuraes de Grficos.

Todos os grficos utilizam HTML5

  • Atributos
    • Tipo de grfico : Permite definir o tipo de grfico que ser exibido por Default caso a opo Grfico Configurvel esteja ativa, ou o tipo de grfico a ser exibido caso a opo esteja desativada.
Barra
Tamanho da fonte : Define o tamanho da fonte no grfico.
Subttulo : Define o subttulo exibido no grfico.
Barra - Orientao : Orientao das barras vertical ou horizontal no grfico.
Barra - Dimenso : Dimenso das Barras (2D ou 3D) no grfico.
Barra - Empilhamento : Empilha as Barras em uma nica Barra (Em srie).
Barra - Agrupamento de sries : Agrupa o grfico de Barras em srie.
Barra - Valores ao lado : Exibe os valores ao lado das Barras.
Barra - Largura : Pode-se definir a largura entre 20 e 70.
Barra - Orientao do valor : Define-se a orientao do grfico, podendo coloca-lo na vertical.
Barra - Posicionamento do valor : Pode-se esolher onde o valor ir se posicionar.
Pizza
Tamanho da fonte : Define o tamanho da fonte no grfico.
Subttulo : Define o subttulo exibido no grfico.
Pizza - Formato : Formato de Pizza ou Donut.
Pizza - Dimenso : Dimenso do grfico de Pizza.
Pizza - Ordenao : Ordenao do grfico Pizza.
Pizza - Formatao de Valores : Formatao dos dados exibidos.
Linha
Tamanho da fonte : Define o tamanho da fonte no grfico.
Subttulo : Define o subttulo exibido no grfico.
Linha - Forma : Formato da Linha.
Linha - Series Group : Linha grouping type.
Area
Tamanho da fonte : Define o tamanho da fonte no grfico.
Subttulo : Define o subttulo exibido no grfico.
Area - Forma : Formato da rea exibida no grfico.
Area - Empilhamento : Empilha as areas no grfico.
Area - Agrupamento de sries : Agrupa o grfico em srie.
Gauge
Tamanho da fonte : Define o tamanho da fonte no grfico.
Subttulo : Define o subttulo exibido no grfico.
Gauge - Forma : Formato de exibio do grfico.
Radar
Tamanho da fonte : Define o tamanho da fonte no grfico.
Subttulo : Define o subttulo exibido no grfico.
Funil
Tamanho da fonte : Define o tamanho da fonte no grfico.
Subttulo : Define o subttulo exibido no grfico.
Funil - Dimenso : Dimenso do Funil (2D ou 3D) no grfico.
Pirmide
Tamanho da fonte : Define o tamanho da fonte no grfico.
Subttulo : Define o subttulo exibido no grfico.
Pirmide - Dimenso : Dimenso (2D ou 3D) no grfico.
Pirmide - Formatao de Valores : Formatao dos dados exibidos.
Pirmide - Forma : Formato de exibio do grfico.
  • Opes em comum
    • Tipo de grfico : Selecione o tipo de grfico entre analtico ou sinttico.
      • Grficos analticos e Grficos sintticos so formas diferentes de lidar com mltiplos grupos pelos critrios de criao dos grficos.
      • Grficos sintticos devem restringir-se ao primeiro grupo pela condio e permitir que o usurio use os links para ver detalhes do prximo grupo de critrios, aumentando o nvel de detalhe.
      • Grficos analticos surgem com todo o grupo por critrios, criando um grfico detalhado completo que no permite a vinculao de outros grficos detalhados.
    • Criar link no grfico : Permite que os grficos tenham um link em seus elementos para aplicaes de grid detalhada ou grficos. Os dados mostrados sero relativos ao valor clicado no grfico.
    • Exibir Valores : Exibe os valores do grfico gerado.
    • Eixo do grfico total geral : Opo para exibir o grfico do total geral como coluna ou linha..
    • Exibir eixo Y com Zero : Forar exibio do valor zero no eixo Y.
    • Largura do Grfico : Largura em pixels dos grficos gerados.
    • Altura do Grfico : Altura em pixels dos grficos gerados.
    • Ordenao dos valores : Ordena os valores dos grficos.
    • Altura para legenda : Altura em porcentagem entre o grfico e a margem superior reservado para as legendas.
    • Modo de exibio dos grficos : Define se os grficos sero mostrados em uma nova janela, lado a lado com o aplicativo, ou carregados na mesma janela do navegador.

Cabealho

Neste bloco feita a definio dos conteudos variveis que faro parte do cabealho.

Configurao do cabealho do resumo

Esta tela pode variar dependendo do formato de cabealho escolhido dentro da pasta Visualizao.

  • Exibir Cabealho : Esta flag determina se o cabealho ser exibido.
  • Ttulo da Consulta : Permite informar um ttulo para ser exibido na aplicao.
  • Variveis de Cabealho : Os campos variveis podem ser preenchidos com qualquer um dos tipos exibidos no Combobox. Dependendo do tipo, ser necessrio associar um contedo ao mesmo. Abaixo descrevemos os tipos existentes:
    • Campo : Quando se escolhe a opo do tipo Campo, abrir um Combobox ao lado com os campos que fazem parte do Select. Escolhendo um desses campos, estar associando o valor do campo para exibio no cabealho.
    • Ttulo : Esta opo quando selecionada exibir no cabealho o valor preenchido no Ttulo do Filtro.
    • Data : Quando selecionado o tipo Data, ser exibido no cabealho da controle a data do sistema no formato dd/mm/aaaa. Existem diversos formatos de exibio utilizando a data e a hora do servidor. O formato poder ser informado no campo texto que aparece ao lado. Para ter acesso aos formatos existentes clique em e aparecer uma tela explicativa.
    • Imagem : Quando selecionado o campo tipo imagem, aparece um campo para o preenchimento do nome da imagem existente no servidor. Para localizar as imagens existentes e selecionar uma, clique no cone Escolher imagem e para disponibilizar novas imagens no servidor clique em Fazer upload de uma imagem .
    • Valor : Quando selecionado o tipo Valor, o contedo preenchido no campo texto que aparece ao lado ser exibido no cabealho, pode ser informado textos e Variveis globais. Ex: Nome do Funcionrio: [v_nome].

Rodap

Configurao do rodap do resumo

Esta tela pode variar dependendo do formato de Rodap escolhido dentro da pasta Visualizao.

  • Exibir Rodap : Esta flag determina se o Rodap ser exibido na aplicao.
  • Variveis de Rodap : Os campos variveis podem ser preenchidos com qualquer um dos tipos exibidos no Combobox, dependendo do tipo ser necessrio associar um contedo ao mesmo. Abaixo descrevemos os tipos existentes:
    • Campo : Quando se escolhe a opo do tipo Campo, abrir um Combobox ao lado com os campos que fazem parte do Select. Escolhendo um desses campos, estar associando o valor do campo para exibio no Rodap.
    • Data : Quando selecionado o tipo Data, ser exibido no Rodap da aplicao a data do sistema no formato dd/mm/aaaa. Existem diversos formatos de exibio utilizando a data e a hora do servidor. O formato poder ser informado no campo texto que aparece ao lado. Para ter acesso aos formatos existentes clique em e aparecer uma tela explicativa.
    • Imagem : Quando selecionado o campo tipo imagem, aparece um campo para o preenchimento do nome da imagem existente no servidor. Para localizar as imagens existentes e selecionar uma, clique no cone Escolher imagem e para disponibilizar novas imagens no servidor clique em Fazer upload de uma imagem.
    • Valor : Quando selecionado o tipo Valor, o contedo preenchido no campo texto que aparece ao lado ser exibido no Rodap. Pode ser informado textos e Variveis globais. Ex: Nome do Funcionrio: [v_nome].
